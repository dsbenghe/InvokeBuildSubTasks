# InvokeBuildSubTasks

`deploy` subtasks

    .\root.build.ps1 deploy [tab]

    task1
    deploy-task2
    .

`build` subtasks

    .\root.build.ps1 build [tab]

    task1
    build-task2
    .

`deploy` sub parameters

    .\root.build.ps1 deploy task1 -[tab]

    RootParam1
    DeployParam1
    DeployParam2

`build` sub parameters

    .\root.build.ps1 build task1 -[tab]

    RootParam1
    BuildParam1
    BuildParam2
